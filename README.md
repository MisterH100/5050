# 5050 - v0.1.0
A simple offline savings challenge app

![Screenshot for 5050 App homepage](./assets/5050v0.1.0.png)

## Why
5050 is a challenge to test your saving habits.
It is offline.

## Features
- Stash your money, choose how much per day.
- From R1 - R100 can be stashed.
- View your progress in %.
- Notice how many days missed.

## How it works
- Choose amount between 1 - 100 to stash, only two choices per day.
- Number chosen will be permanently removed from the list.
- Progress increases torwads R5050.
- Days are tracked on each stash.

## Installation/Usage
- Open web link
- Works offline

- HTML
